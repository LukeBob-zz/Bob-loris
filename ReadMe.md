# Bob-Loris

**Setup**

        $ git clone https://github.com/LukeBob/Bob-loris.git
 
        $ cd Bob-loris
 
        $ chmod 755 Bob-loris.py
 
        $ ./Bob-loris.py -H <host> -P <port> -M <max-sockets Default=150>
    
**Example**
          
          $ ./Bob-loris.py -H 192.168.1.160 -P 80 -M 149
